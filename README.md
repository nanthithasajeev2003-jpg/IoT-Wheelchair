# IoT-Wheelchair
IoT based wheelchair with gesture and voice control and health monitoring
##  Overview
This project implements a smart wheelchair system using gesture control, voice commands, and IoT-based health monitoring.
##  Features
- Gesture-based movement using accelerometer
- Voice control system
- Health monitoring (Temperature, SpO2, Pulse rate)
- IoT monitoring using Blynk
- Wireless communication using ESP32 & NodeMCU

##  Hardware Used
- ESP32
- NodeMCU (ESP8266)
- Arduino UNO
- L298N Motor Driver
- Pulse Sensor
- DS18B20 Temperature Sensor
- MAX30102 (SpO2)

##  Software Used
- Arduino IDE
- Blynk App
## 📊 Working
- Accelerometer detects hand gesture
- NodeMCU sends data via WiFi
- ESP32 controls motors
- Sensors monitor health data
- Data uploaded to IoT platform
